HTML, CSS, JavaScript(jQuery)

使用環境

aws, GitHub

作業範囲

全て(デザイン、コーディング)

制作期間

9日間(2021/02/09〜02/17)

制作の目的

ページ訪問者がそのお店に行きたくなるようなサイトを制作すること。

制作のポイント

  • レスポンシブWebデザイン(ブレイクポイントは1つ)を取り入れています。

  • スライダー(画像)をトップ全面に配置し、インパクトを出しています。
  • PC版では、スライダー(画像)上に目印を配置することで、スクロールしてもらいやすくしています。
  • 見出しがフェードインすることで、ページ訪問者に飽きさせない工夫をしています。

制作の目的

食べたことのある食材やアレルギーの記録に役立てること。離乳食を記録するだけでなく、栄養に関する簡易説明のページを設けることで、親の食知識を増やせるようにしています。

  • レスポンシブWebデザイン(ブレイクポイントは1つ)を取り入れています。
  • 栄養に関する簡易説明のページでは、checkボタンをクリックすると文字が現れるようになっており、ユーザーが見たい情報に集中できるよう工夫しました。
  • 離乳食記録ページでは、食べた食材の情報を一覧表示で確認できます。
